javascript学习笔记1

arguments的使用

  • 不确定实参个数时，使用arguments获取所有参数

arguments并不是真正意义上的数组：

  1. 具有length

  2. 以索引进行存储

  3. 没有pop()与push()等数组方法

javascript作用域

  • javascript作用域分为全局作用域与局部作用域

全局作用域

  • 整个script标签可用,或单独的js文件可用

局部作用域

  • 仅在函数内部起作用

在函数内部未经声明，直接赋值的变量也是全局变量

变量的销毁

  • 全局变量：浏览器关闭时才会销毁。

  • 局部变量：代码块结束后，就会被销毁。

作用域链

当函数中嵌套了函数，作用域链就产生了。

js运行步骤

  • javascript解析器在运行javascript语句时分两步：预解析和代码执行。

预解析

  • 预解析分为变量预解析（变量提升）和函数预解析（函数提升）。

  • js解析器会将所有的var与function提升到当前作用域的最前面。

代码执行

  • 代码按照书写顺序，由上到下依次执行

new的执行过程

  1. 在内存中创建空对象。

  2. this指向空对象。

  3. 执行构造函数，并添加属性和方法。

  4. 返回新对象。

事件对象

  • 只有有onclick事件绑定才会有事件对象产生。

  • 事件对象是事件的一系列相关数据的集合。
